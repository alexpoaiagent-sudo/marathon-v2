# Методология для участников

Здесь собираем пояснения, которые дополняют основные материалы марафона:

- развёрнутые инструкции по работе с трекером и экзокортексом;
- примеры формулировок рабочих продуктов;
- советы по использованию ИИ в каждом состоянии (Хаос, Тупик, Поворот);
- ответы на частые вопросы, которые всплывают после встреч.

Каждый документ должен быть коротким и ссылаться на основную страницу, откуда участник пришёл (встречу или практику).
