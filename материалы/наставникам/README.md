# Папка наставника

Здесь всё, что помогает фасилитаторам вести марафон:

- используй сценарии встреч из `../участникам/встречи/` как основной сценарий;
- уточняй терминологию в `../участникам/понятия.md` и `../участникам/состояния-и-методы.md`;
- выкладывай свои чек-листы и адаптации в `методология/`.

Перед стартом марафона пробегись по этой папке и добавь свои заметки, чтобы команда наставников работала единообразно.
