# Материалы Марафона 2.0

Все рабочие документы для команды, участников и наставников с понятными русскими названиями.

## Как устроено

- **участникам/** — всё, что получает участник: описание, опросник, встречи, шаблоны и примеры.
- **наставникам/** — опорные материалы для фасилитаторов и ведущих.

## Где хранить методологию

- Дополнительные пояснения для участников складывайте в **участникам/методология/**.
- Методички, чек-листы и адаптации под аудиторию для наставников — в **наставникам/методология/**.

Так команда сразу понимает, кому предназначен документ и куда его положить.
