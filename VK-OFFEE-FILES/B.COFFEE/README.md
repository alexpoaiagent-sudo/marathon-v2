# База знаний сети «Вкусный Кофе»

**Хранилище знаний по методу SRT** (Systems-Roles-Table) — организация проекта через таблицу 3×3 систем и ролей с интеграцией First Principles Framework (FPF).

## Для кого эта база

- **Управляющий сети** — стратегия, развитие, контроль
- **Бариста** — операционные процессы, стандарты, обучение
- **Официанты** — сервис, чек-листы, должностные инструкции
- **Партнёры и франчайзи** — модель бизнеса, стандарты бренда
- **AI-агенты** — структурированные знания для автоматизации

## Что такое SRT-метод

SRT (Systems-Roles-Table) — это способ организации знаний через **таблицу 3×3**, которая определяет **9 семейств документов** + **1 метасемейство** (управление хранилищем).

### Таблица 3×3: 9 семейств документов

```
                     Предприниматель  Инженер           Менеджер
                     (Зачем?)         (Как устроено?)   (Как работает?)
┌────────────────────────────────────────────────────────────────────┐
│ HoReCaMarket       F1               F2                F3           │
│ (Надсистема)       Контекст         Окружение         Партнёрства  │
│                    и рынок          и интерфейсы      и франшиза   │
├────────────────────────────────────────────────────────────────────┤
│ VkusnyCoffee       F4               F5                F6           │
│ (Целевая система)  Ценности         Меню и продукты   Операции     │

├────────────────────────────────────────────────────────────────────┤

│                    и принципы       и инструменты     и обучение   │
└────────────────────────────────────────────────────────────────────┘
```

## Три системы проекта

### System-of-Interest (Целевая система)
**Сеть кофеен «Вкусный Кофе»** — то, что мы создаём.

Переводим гостя из состояния неудовлетворённости (усталость, голод) в состояние удовлетворения (бодрость, комфорт).

### Suprasystem (Надсистема)
**Городская среда и рынок HoReCa** — система, в которую встроены кофейни.

Формирует поток гостей, конкуренцию, условия работы.

### Constructor (Система создания)
**Команда управления** — система, которая создаёт и развивает сеть.

Люди, инструменты, методология, ресурсы.

## Ключевые метрики

### Гостевой опыт
- **P95 ≤ 4:00** — 95% напитков готовы за 4 минуты
- **NPS** — индекс лояльности
- **CRR** — доля повторных визитов

### Экономика
- **AOV** — средний чек
- **Фудкост %** — себестоимость / выручка
- **Выручка/час** — производительность

### Команда
- **Часы/₽** — производительность персонала
- **Текучесть** — стабильность команды

## Ценности

- **Порядочность** — никакого воровства
- **Искренность и открытость** — идти навстречу в разговоре
- **Стрессоустойчивость** — позитивный настрой обязателен
- **Ответственность** — доводить начатое до результата
- **Доброжелательность** — вежливость со всеми гостями
