# Финансовые расчеты

Расчеты себестоимости и калькуляции напитков. Для чтения используйте Markdown-версии, оригиналы остаются как источники.

## Что внутри
Markdown-версии расчётов и исходные файлы для пересчётов.

## Кому нужно
- Управляющему и операционному менеджеру.
- Финансисту при проверках маржинальности.

## Частые задачи
- Проверить актуальные цены — откройте [себестоимость в Markdown](./drink-costing.md).
- Пересчитать показатели — редактируйте [XLSX исходник](./drink-costing.xlsx).
- Подготовить презентацию или обучение — используйте [калькуляции напитков](./drink-calculations.md).
