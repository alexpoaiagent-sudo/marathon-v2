## Как добавить в GitHub

```bash
# 1. Распакуй архив в папку репозитория
unzip vk-offee-export.zip -d /path/to/VK-offee/

# 2. Перейди в репозиторий
cd /path/to/VK-offee/

# 3. Добавь файлы
git add .

# 4. Закоммить
git commit -m "docs: перенос документации из Google Drive

- Должностная инструкция повара (КУХНЯ)
- Вакансии бариста, помощника, раннера (ПЕРСОНАЛ)
- Дорожная карта операционного менеджера
- Карточка ИП (реквизиты)"

# 5. Запушь
git push
```
