# AGENTS

## Общие правила
- Работаем только через pull request.
- Сначала готовим план, затем выполняем изменения.
- Не трогаем CI и секреты.
- Не переименовываем и не удаляем файлы без явного указания.
- Противоречия не "чинить молча" — фиксируем их в отчёте и предлагаем вариант решения.
- Определения метрик и терминов выносить в `glossary.md`.

## Как запускать задачи для агента
Используйте следующий шаблон запроса:

```
Задача: <краткое описание цели>
Контекст/файлы: <пути или ссылки на важные части проекта>
Ограничения: <дедлайны, запреты, ожидания по тестам>
Ожидаемый результат: <что должно быть в итоговом PR/коммите>
```
