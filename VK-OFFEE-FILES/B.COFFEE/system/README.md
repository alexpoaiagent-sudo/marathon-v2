# Система

## Что внутри
Системные описания сети, глоссарий терминов, ценности и карточки контроля качества.

## Кому нужно
- Управляющему сети и операционному менеджеру.
- HR и наставникам, которые формируют стандарты.
- Всем, кто обновляет светофор и единые определения.
