# F7: Принципы и экономика

**Роль: Предприниматель (Зачем?)**

Этот раздел описывает **бизнес-принципы и экономику** системы создания.

## Документы

### [finance/](finance/)
Финансовые расчёты и калькуляции.

**Содержание:**
- Себестоимость напитков
- Калькуляции блюд
- Маржинальность по категориям
- Фудкост и списания

## Ключевые принципы

### Экономическая модель
- **AOV (средний чек)** — выручка / количество чеков
- **Фудкост %** — себестоимость / выручка × 100
- **Часы/₽** — отработанные часы / выручка

### Бизнес-метрики
- Выручка по локациям
- Маржинальность по категориям
- Оборачиваемость запасов
- Списания и потери

### Инвестиционная привлекательность
- Окупаемость точек
- Масштабируемость модели
- Франчайзинговый потенциал

## Финансовые процессы

1. **Планирование** — бюджеты, прогнозы
2. **Учёт** — себестоимость, выручка
3. **Контроль** — отклонения, корректировки
4. **Анализ** — рентабельность, точки роста
