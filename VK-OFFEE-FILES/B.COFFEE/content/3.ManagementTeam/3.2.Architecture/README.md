# F8: Платформа и инструменты

**Роль: Инженер (Как устроено?)**

Этот раздел описывает **инструменты и системы**, используемые для создания и поддержания работы сети кофеен.

## Документы

### [vendors/](vendors/)
Поставщики и прайс-листы.

**Содержание:**
- Контакты поставщиков
- Прайс-листы
- Каталоги продукции
- Условия поставок

## Ключевые инструменты

### Операционные системы
- **POS (Point-of-Sale)** — касса для приёма заказов
- **KDS (Kitchen Display System)** — экран заказов для кухни и бара
- **Учётная система** — складской учёт, инвентаризации

### Оборудование
- Кофемашины и гриндеры
- Холодильное оборудование
- Кухонное оборудование
- Лёдогенераторы

### Поставщики
- Кофе и чай
- Молочная продукция
- Выпечка и десерты
- Расходные материалы

## Процессы снабжения

1. **Планирование** — прогноз потребности
2. **Заказ** — формирование заявок
3. **Приёмка** — контроль качества и количества
4. **Хранение** — FIFO, температурные режимы
