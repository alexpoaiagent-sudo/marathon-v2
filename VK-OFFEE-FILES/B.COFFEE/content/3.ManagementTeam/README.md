# F7-F9: Система создания — Команда управления

**Система создания (Constructor)** — это система, которая создаёт целевую систему. Для сети кофеен это **команда управления и развития**.

Система создания включает людей, инструменты, методологию и ресурсы, необходимые для создания и поддержания работы сети кофеен.

## Ключевые роли

- **Управляющий сети** — стратегия, развитие, контроль
- **Бариста** — приготовление напитков, сервис
- **Официант/раннер** — обслуживание зала
- **Повар** — приготовление блюд
- **Наставник** — обучение новых сотрудников

## Ключевые процессы

1. **Подбор и обучение** — найм, адаптация, развитие
2. **Снабжение** — работа с поставщиками, закупки
3. **Контроль качества** — аудиты, метрики, улучшения
4. **Финансовый учёт** — себестоимость, выручка, прибыль
