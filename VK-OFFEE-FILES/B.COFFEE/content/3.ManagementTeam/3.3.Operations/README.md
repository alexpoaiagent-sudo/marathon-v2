## Ключевые роли

### Бариста
- Приготовление напитков
- Сервис и коммуникация
- Поддержание чистоты
- Контроль качества

### Официант/Раннер
- Обслуживание зала
- Приём заказов
- Контроль смены
- Работа с кассой

### Управляющий
- Планирование смен
- Контроль метрик
- Работа с поставщиками
- Обучение команды

## Процессы управления командой

### 1. Подбор
- Собеседования
- Проверка ценностей
- Тестовые смены

### 2. Адаптация
- План адаптации
- Наставничество
- Контрольные точки

### 3. Обучение
- Базовые навыки
- Стандарты сервиса
- Продуктовое обучение

### 4. Развитие
- Карьерные треки
- Повышение квалификации
- Кросс-тренинг

## Метрики команды

- **Часы/₽** — производительность
- **Текучесть** — стабильность команды
- **Скорость адаптации** — время до полной самостоятельности
- **Качество сервиса** — NPS, отзывы
