# F4-F6: Целевая система — Сеть кофеен «Вкусный Кофе»

**Целевая система (System-of-Interest)** — это то, что мы создаём. Для нашего проекта это **сеть кофеен «Вкусный Кофе»**.

Целевой системой является система **«гость в удовлетворённом состоянии»** — мы переводим гостя из состояния неудовлетворённости (усталость, голод, жажда) в состояние удовлетворения (бодрость, сытость, комфорт).

## Подсистемы

Сеть состоит из отдельных кофеен:
- **Кофейня на ул. Самокиша** — центр города, туристы и офисные
- **Кофейня на ул. Тургенева** — спальный район, высокий трафик
- **Кофейня на ул. Луговой** — семейная аудитория, двухэтажная
- **Проект в парке** — новое строительство (в разработке)

## Ключевые метрики

- **P95 времени выдачи** — скорость обслуживания
- **AOV** — средний чек
- **CRR** — доля повторных визитов
- **NPS** — индекс лояльности
