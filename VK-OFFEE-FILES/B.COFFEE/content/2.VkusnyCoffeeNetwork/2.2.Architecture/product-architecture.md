# F5: Архитектура продуктового предложения

**Семейство:** F5 (Целевая система × Инженер)
**Вопрос:** Как устроен продукт?

## Назначение документа

Этот документ описывает **структуру продуктового предложения** сети кофеен: меню, рецептуры, стандарты качества.

## Структура меню

### Принципы построения меню

1. **Короткое меню** — быстрые позиции для высокой скорости (P95 ≤ 4:00)
2. **Бандлы** — готовые комбинации под сценарии
3. **Сезонность** — адаптация под лето/зиму
4. **Локальность** — учёт особенностей локации

## Архитектура напитков

### Эспрессо-based напитки

**Базовый компонент:** Эспрессо (двойной шот, 18г кофе, 36г напитка, 25-30 сек)

**Структура:**

| Напиток | Эспрессо | Молоко | Вода | Лёд | Время |
|---------|----------|--------|------|-----|-------|
| **Эспрессо** | 18г | - | - | - | ≤ 2:00 |
| **Американо** | 18г | - | 120мл | - | ≤ 2:30 |
| **Капучино** | 18г | 120мл | - | - | ≤ 3:30 |
| **Латте** | 18г | 180мл | - | - | ≤ 3:30 |
| **Флэт уайт** | 18г | 150мл | - | - | ≤ 3:30 |
| **Айс латте** | 18г | 150мл | - | 100г | ≤ 4:00 |

**Стандарты качества:**
- Температура эспрессо: 92-96°C
- Температура молока: 60-65°C
- Пенка молока: микропена, глянцевая
- Латте-арт: базовый (сердце, розетта)

### Фильтр-кофе

**Методы заваривания:**

| Метод | Кофе | Вода | Время | Выход |
|-------|------|------|-------|-------|
| **V60** | 15г | 250мл | 2:30 | 220мл |
| **Аэропресс** | 15г | 200мл | 2:00 | 180мл |
| **Batch brew** | 60г | 1000мл | 4:00 | 900мл |

**Стандарты:**
- Температура воды: 92-96°C
- Помол: средний (для V60), средне-мелкий (для аэропресс)
- TDS: 1.2-1.5%

### Холодные напитки

**Структура:**

| Напиток | База | Добавки | Лёд | Время |
|---------|------|---------|-----|-------|
| **Колд брю** | 200мл | - | 100г | ≤ 2:00 |
| **Айс латте** | Эспрессо 18г | Молоко 150мл | 100г | ≤ 4:00 |
| **Фраппе** | Эспрессо 18г | Молоко 100мл, сироп | 150г | ≤ 5:00 |
| **Лимонад** | Сироп 30мл | Вода 200мл, лимон | 100г | ≤ 3:00 |

**Стандарты:**
- Лёд: чистый, без запаха
- Температура: 4-8°C
- Подача: в прозрачном стакане

## Архитектура еды

### Выпечка

**Поставщики:**
- Goryachie Sloechki — слойки, кулебяки, кексы
- Melnitsa — хлеб, багеты
- Sev Sloyki — слоёные изделия

**Стандарты:**
- Свежесть: день в день или -1 день
- Температура подачи: комнатная или подогретая (по запросу)
- Упаковка: индивидуальная

**Хранение:**
- Витрина: комнатная температура, закрытая
- Срок: до конца дня
- Остатки: списание или скидка в конце дня

### Горячие блюда

**Структура:**

| Блюдо | Компоненты | Время приготовления | Температура подачи |
|-------|------------|---------------------|-------------------|
| **Сэндвич** | Хлеб, начинка, соус | ≤ 5:00 | 60-70°C |
| **Панини** | Чиабатта, начинка | ≤ 7:00 | 70-80°C |
| **Суп** | Бульон, овощи, мясо | ≤ 3:00 (разогрев) | 75-85°C |

**Стандарты:**
- Температура: проверка термометром
- Подача: на тарелке/в миске
- Сервировка: салфетки, приборы

### Десерты

**Типы:**
- **Чизкейки** — порционные, охлаждённые
- **Тарты** — порционные, комнатная температура
- **Моти** — японские рисовые шарики с начинкой
- **Макарунс** — французские миндальные печенья

**Стандарты:**
- Хранение: холодильник (чизкейки) или витрина (остальное)
- Подача: на тарелке с вилкой
- Срок годности: указан на упаковке

## Бандлы (готовые комбинации)

### Утренний бандл «Быстрый старт»
**Состав:** Американо + круассан
**Цена:** 250₽ (скидка 10%)
**Целевая аудитория:** Офисные сотрудники
**Время:** 7:30-10:00

### Обеденный бандл «Перекус»
**Состав:** Капучино + сэндвич
**Цена:** 350₽ (скидка 10%)
**Целевая аудитория:** Офисные сотрудники
**Время:** 12:00-14:00

### Семейный бандл «Вместе»
**Состав:** 2 латте + 2 десерта + какао
**Цена:** 900₽ (скидка 15%)
**Целевая аудитория:** Семьи с детьми
**Время:** 17:00-20:00

### Рабочий бандл «Фокус»
**Состав:** Фильтр-кофе (большой) + печенье
**Цена:** 200₽ (скидка 10%)
**Целевая аудитория:** Фрилансеры
**Время:** Весь день

## Сезонное меню

### Летнее меню (май-сентябрь)

**Акцент:** Холодные напитки

**Специальные позиции:**
- Колд брю с тоником
- Айс матча латте
- Фруктовые лимонады (клубника, маракуйя, манго)
- Смузи-боулы

**Изменения:**
- Увеличение доли холодных напитков до 60%
- Лёгкие десерты (фруктовые тарты)
- Холодные супы (гаспачо)

### Зимнее меню (октябрь-апрель)

**Акцент:** Горячие напитки и согревающие позиции

**Специальные позиции:**
- Раф-кофе
- Горячий шоколад с маршмеллоу
- Пряный латте (с корицей и кардамоном)
- Глинтвейн безалкогольный

**Изменения:**
- Увеличение доли горячих напитков до 80%
- Сытные десерты (чизкейки, брауни)
- Горячие супы

## Стандарты качества

### Контроль качества кофе

**Параметры:**
- **Зерно:** specialty grade, Q-score > 80
- **Свежесть обжарки:** < 2 недель
- **Помол:** свежий, перед приготовлением
- **Вода:** фильтрованная, TDS 50-150 ppm
- **Температура:** 92-96°C

**Проверка:**
- Калибровка эспрессо: каждое утро
- Дегустация: первый шот смены
- Корректировка помола: при необходимости

### Контроль качества молока

**Параметры:**
- **Жирность:** 3.2%
- **Температура хранения:** 2-6°C
- **Срок годности:** > 5 дней до истечения
- **Температура взбивания:** 60-65°C

**Проверка:**
- Проверка срока годности: при приёмке
- Температура холодильника: ежедневно
- Качество пенки: каждый напиток

### Контроль качества еды

**Параметры:**
- **Свежесть:** день в день или -1 день
- **Температура хранения:** согласно СанПиН
- **Температура подачи:** согласно стандартам
- **Внешний вид:** без повреждений

**Проверка:**
- Приёмка: проверка свежести и внешнего вида
- Хранение: контроль температуры
- Подача: проверка температуры термометром
