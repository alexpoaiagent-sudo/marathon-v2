## Ключевые направления

### Франчайзинг
- Модель франшизы

- Поддержка франчайзи

- **Школы и кружки** — купоны для учеников
- **Офисы** — корпоративные абонементы
- **Салоны красоты** — кросс-промо

- Санитарные проверки
- Пожарная безопасность
- Лицензирование
- Налоговая отчётность

- Отзывы и рейтинги
- Работа с блогерами

### С арендодателями
- Условия аренды
- Режим работы
- Ремонт и обслуживание

### С управляющими компаниями
- Согласование вывесок
- Летние площадки

### С местным сообществом
- Участие в районных мероприятиях

- Благотворительность
- Формирование лояльности
