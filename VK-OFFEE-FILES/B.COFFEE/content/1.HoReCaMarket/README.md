# F1-F3: Надсистема — Городская среда и рынок HoReCa

**Надсистема** — это система, частью которой становится целевая система (сеть кофеен «Вкусный Кофе»).

Для сети кофеен надсистемой является **городская инфраструктура и рынок HoReCa**, которая формирует условия работы, поток гостей, конкуренцию и возможности для развития.

## Ключевые компоненты надсистемы

| Компонент | Влияние на сеть кофеен |
|-----------|------------------------|
| Городская инфраструктура | Физическая доступность для гостей |
| Жилая и деловая среда | Формирует основной поток гостей |
| Социальная инфраструктура | Создаёт устойчивые маршруты |
| Торговая инфраструктура | Генерирует дополнительный трафик |
| Конкурентная среда | Формирует ожидания по ценам и качеству |
