# F0: Метасемейство — Управление хранилищем

**F0 — это 10-е семейство**, которое управляет остальными девятью. В отличие от F1-F9, метасемейство **не делится по ролям** Предприниматель/Инженер/Менеджер, а организовано по функциональным областям.

## Назначение

Метасемейство обеспечивает:
- Единую терминологию и понятийный аппарат
- Правила работы с документами
- Навигацию по хранилищу
- Контроль качества и актуальности

## Целевая аудитория

- AI-агенты
- Редакторы хранилища
- Администраторы проекта
