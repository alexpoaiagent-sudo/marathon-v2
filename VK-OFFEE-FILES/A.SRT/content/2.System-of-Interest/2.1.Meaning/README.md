# Предприниматель: Смысл целевой системы

## Назначение раздела

Здесь документируются требования к продукту, его ценность и цели с точки зрения пользователей.

## Что здесь размещается

- **Требования пользователей** — что нужно пользователям
- **Функциональные требования** — что должна делать система
- **Нефункциональные требования** — качественные характеристики
- **Use cases** — сценарии использования
- **Ценностное предложение** — какую ценность даёт продукт
