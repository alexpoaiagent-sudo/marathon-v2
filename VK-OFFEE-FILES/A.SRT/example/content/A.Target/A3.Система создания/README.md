# A3. Система создания — ORA (ивент-продукт)

## Назначение

Документация о **системе создания впечатлений** — мероприятии как продукте.

## Определение

**ORA** — система (мероприятие-как-продукт), которая в конкретном проведении воздействует на человека внутри "среды" и переводит его в состояние "впечатлённости".

## Важно: описания ≠ части системы

Следующее — это **описания ORA** (пакеты описаний, по которым ORA проектируют и проводят), а НЕ части ORA как системы:

- Время/место
- Программа и активности
- Правила участия
- Сценарий взаимодействий
- Коммуникации до/во время/после
- Критерии качества
- Плейбуки и чек-листы
