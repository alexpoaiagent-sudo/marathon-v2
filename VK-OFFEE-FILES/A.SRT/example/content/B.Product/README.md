# Ядро B: Product — Наша система

## Назначение

Документация о создании **Приложения** для поддержки ORA.

## Системы ядра

| Код | Система | Вопрос | Описание |
|-----|---------|--------|----------|
| **B1** | Создатель ORA | Для кого? | Надсистема: организатор + площадка + инструменты |
| **B2** | Приложение | Что создаём? | Целевая система: платформа для ORA |
| **B3** | Planora | Кто создаёт? | Система создания: ИТ-служба + ORA-студия |

## Структура

```
B.Product/
├── B1.Надсистема/               # Создатель ORA
│   ├── B1.1.Предприниматель/    # Ценность для организатора
│   ├── B1.2.Инженер/            # Структура создателя ORA
│   └── B1.3.Менеджер/           # Workflow создателя
│
├── B2.Целевая система/          # Приложение
│   ├── B2.1.Предприниматель/    # Требования
│   ├── B2.2.Инженер/            # Архитектура
│   └── B2.3.Менеджер/           # Использование
│
└── B3.Система создания/         # Planora
    ├── B3.1.Предприниматель/    # Миссия
    ├── B3.2.Инженер/            # ИТ + ORA-студия
    └── B3.3.Менеджер/           # Процессы
```

## Связь с ядром A

Приложение (B2) используется в ORA (A3) как инструмент цифрового контура.

→ См. [X.Bridge](../X.Bridge/) для деталей интеграции.

## Для кого это ядро

- **ИТ-служба** — разработчики, DevOps, архитекторы
- **Product Owner** — управление требованиями
- **ORA-студия** — внедрение и обучение
