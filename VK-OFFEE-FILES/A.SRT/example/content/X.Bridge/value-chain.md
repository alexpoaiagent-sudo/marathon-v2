# Цепочка создания ценности

## Полная цепочка

```
Planora → Приложение → Создатель ORA → ORA → Впечатлённый человек → Среда
  (B3)      (B2)          (B1/A3)      (A3)        (A2)             (A1)
```

## Описание каждого звена

### 1. Planora создаёт Приложение

**Вход:** Потребности организаторов ORA
**Выход:** Работающее Приложение
**Ценность:** Инструмент для создания ORA

### 2. Приложение поддерживает Создателя ORA

**Вход:** Приложение как инструмент
**Выход:** Цифровой контур для ORA
**Ценность:** Автоматизация, данные, коммуникации

### 3. Создатель ORA создаёт ORA

**Вход:** Приложение + площадка + контент + команда
**Выход:** Проведённое мероприятие
**Ценность:** Опыт для участников

### 4. ORA трансформирует человека

**Вход:** Человек в исходном состоянии
**Выход:** Впечатлённый человек
**Ценность:** Повышенная впечатлённость

### 5. Впечатлённый человек влияет на среду

**Вход:** Новые впечатления, связи, знания
**Выход:** Изменения в поведении, рекомендации
**Ценность:** Распространение эффекта

## Метрики по цепочке

| Звено | Метрика |
|-------|---------|
| Planora → Приложение | Функциональность, стабильность |
| Приложение → Создатель | Adoption, retention организаторов |
| Создатель → ORA | Количество и качество мероприятий |
| ORA → Человек | Впечатлённость (W, W+Δ) |
| Человек → Среда | Рекомендации, повторное участие |
