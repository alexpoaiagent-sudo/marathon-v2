# Методология наставника

Собираем здесь:

- чек-листы подготовки к каждой встрече (оборудование, тайминг, материалы);
- подсказки по адаптации практик под разные состояния участников;
- риски и типичные затыки, которые нужно отработать заранее;
- примеры формулировок вопросов и комментариев для обратной связи.

Каждый файл держи коротким и начинай с цели: для какой встречи или ситуации предназначена методичка.
