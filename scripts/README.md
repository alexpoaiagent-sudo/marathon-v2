# Служебные скрипты Marathon-v2

Эта папка содержит вспомогательные bash-скрипты для управления и проверки репозитория.

## Скрипты

### `check-integrity.sh`
**Назначение:** Ежедневная проверка целостности репозитория

**Проверяет:**
- Целостность всех markdown файлов
- Наличие YAML front matter в обязательных файлах
- Валидность внутренних ссылок
- Версионирование и CHANGELOG

**Запуск:**
```bash
bash scripts/check-integrity.sh
```

**Генерирует:** Отчёт в `reports/health-check-YYYY-MM-DD.md`

**Периодичность:**
- ✅ Ежедневно в 12:00 (GitHub Actions)
- ✅ При каждом коммите в main
- ✅ Вручную по требованию

---

## Добавление новых скриптов

При добавлении нового скрипта:
1. Убедитесь, что он имеет заголовок с описанием
2. Добавьте описание в этот README
3. Используйте шебанг: `#!/bin/bash`
4. Назначьте права на исполнение: `chmod +x script-name.sh`

---

## Требования

- Bash 4.0+
- Стандартные Unix утилиты (`grep`, `find`, `sed`)
- Для GitHub Actions: Python 3.8+ (если требуется)
