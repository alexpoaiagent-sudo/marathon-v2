# Структура репозитория marathon-2.0

Это описание текущей структуры, чтобы команда быстро ориентировалась в материалах марафона.

---

## Корневая директория

```
marathon-2.0/
├── README.md                # Главная страница с описанием марафона
├── CONTRIBUTING.md          # Руководство по внесению вклада
├── LICENSE                  # Лицензия (CC BY-SA 4.0)
├── STRUCTURE.md             # Описание структуры (этот файл)
├── DICTIONARY.md            # Словарь и правила именования
├── материалы/               # Все материалы на русском
├── reports/                 # Отчёты и диагностические выгрузки
└── scripts/                 # Скрипты автоматизации
```

---

## Директория `материалы/`

Блок с понятными названиями для русскоязычной команды. Делится на два набора: что читать участникам и что использовать наставникам.

```
материалы/
├── README.md                       # Краткая навигация по материалам
├── участникам/                     # Всё, что даём участникам
│   ├── обзор-марафона.md           # Полное описание и философия
│   ├── опросник.md                 # 12 вопросов для диагностики
│   ├── понятия.md                  # Термины и словарь марафона
│   ├── состояния-и-методы.md       # Связка «состояние → метод»
│   ├── минимальные-практики.md     # 6 базовых практик
│   ├── промт-для-ии-агента.md      # Промт для диагностики через ИИ
│   ├── встречи/                    # Сценарии четырёх встреч
│   │   ├── встреча-01.md
│   │   ├── встреча-02.md
│   │   ├── встреча-03.md
│   │   └── встреча-04.md
│   ├── шаблоны/                    # Заполняемые формы
│   │   ├── шаблон-трекера.md
│   │   └── чеклист-стоп-момента.md
│   ├── примеры/                    # Будущие примеры заполнений
│   └── методология/                # Методические пояснения для участников
│       └── README.md
└── наставникам/                    # Материалы для фасилитаторов
    ├── README.md                   # Как работать с папкой наставника
    └── методология/                # Отдельные методички и чек-листы
        └── README.md
```

---

## Навигация для участников

1. Начинают с **материалы/участникам/опросник.md** и определяют состояние.
2. Сверяются с **материалы/участникам/понятия.md**, чтобы говорить на одном языке.
3. Готовятся к встречам, двигаясь по файлам в **материалы/участникам/встречи/**.
4. Пользуются готовыми формами из **материалы/участникам/шаблоны/**.
5. Если нужна дополнительная методика или контекст — смотрят **материалы/участникам/методология/**.

---

## Навигация для наставников

- Базовая ориентация — **материалы/наставникам/README.md**.
- Сценарии встреч и рабочий поток участников лежат в **материалы/участникам/встречи/** — наставники работают с теми же документами.
- Методические подсказки, чек-листы, адаптации под аудиторию добавляются в **материалы/наставникам/методология/**.

---

## Планируемые пополнения

- **материалы/участникам/примеры/** — готовые образцы трекеров и рабочих продуктов.
- **материалы/участникам/методология/** — короткие методички по работе с экзокортексом, трекером и экосистемой инструментов.
- **материалы/наставникам/методология/** — гайды по фасилитации, рискам и типовым вопросам.
