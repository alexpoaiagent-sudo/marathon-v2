# Структура репозитория marathon-2.0

Это описание текущей структуры, чтобы команда быстро ориентировалась в материалах марафона.

---

## Корневая директория

```
marathon-2.0/
├── README.md                # Главная страница с описанием марафона
├── CONTRIBUTING.md          # Руководство по внесению вклада
├── LICENSE                  # Лицензия (CC BY-SA 4.0)
├── STRUCTURE.md             # Описание структуры (этот файл)
├── DICTIONARY.md            # Словарь и правила именования
├── metamaterial/               # Все материалы на русском
├── reports/                 # Отчёты и диагностические выгрузки
└── scripts/                 # Скрипты автоматизации
```

---

## Директория `metamaterial/`

Методологическая база марафона — справочники, понятия, инструменты.

```
metamaterial/
├── README.md                       # Краткая навигация
├── понятия.md                      # Термины и словарь марафона
├── состояния-и-методы.md           # Связка «состояние → метод»
├── минимальные-практики.md         # 6 базовых практик
├── источник-правды/                # Системное описание марафона
├── participants/                   # Инструменты для участников
│   ├── README.md
│   └── templates/                  # Шаблоны для работы
│       ├── tracker.md              # Шаблон трекера
│       └── stop-moment-checklist.md # Чеклист стоп-момента
└── mentor/                         # Инструменты для наставников
    ├── README.md
    └── prompt-for-ai-agent.md      # Промт для ИИ-диагностики (единый источник)
```

## Директория `materials/`

Линейные методические материалы для работы (читаются последовательно).

```
materials/
├── participants/                   # Материалы для участников
│   ├── README.md
│   ├── 00_questionnaire.md         # Опросник
│   ├── 01_meeting_1.md             # Встреча 1
│   ├── 02_meeting_2.md             # Встреча 2
│   ├── 03_meeting_3.md             # Встреча 3
│   └── 04_meeting_4.md             # Встреча 4
└── mentor/                         # Материалы для наставников
    ├── README.md
    ├── 01_questionnaire_mentor.md  # Опросник (методичка)
    ├── 01_meeting_1_mentor.md      # Встреча 1 (методичка)
    ├── 02_meeting_2_mentor.md      # Встреча 2 (методичка)
    ├── 03_meeting_3_mentor.md      # Встреча 3 (методичка)
    ├── 04_meeting_4_mentor.md      # Встреча 4 (методичка)
    └── methodology/                # Методологические материалы
```

---

## Навигация для участников

1. Начинают с **materials/participants/00_questionnaire.md** и определяют состояние
2. Сверяются с **metamaterial/понятия.md**, чтобы говорить на одном языке
3. Готовятся к встречам, двигаясь по файлам в **materials/participants/** (01-04_meeting)
4. Пользуются готовыми шаблонами из **metamaterial/participants/templates/**
5. Изучают методологию в **metamaterial/** (состояния-и-методы, минимальные-практики)

---

## Навигация для наставников

- Базовая ориентация — **materials/mentor/README.md**
- Сценарии встреч и методички лежат в **materials/mentor/** (01-04_meeting_mentor)
- Инструменты диагностики и промты — в **metamaterial/mentor/**
- Методологическая база — в **metamaterial/** (понятия, состояния, источник-правды)

---

## Планируемые пополнения

- **metamaterial/участникам/примеры/** — готовые образцы трекеров и рабочих продуктов.
- **metamaterial/участникам/методология/** — короткие методички по работе с экзокортексом, трекером и экосистемой инструментов.
- **metamaterial/наставникам/методология/** — гайды по фасилитации, рискам и типовым вопросам.
